# Sound Credits

This mod includes sound effects sourced from the **SoundMonster** repository:
- https://github.com/jonjonsson/SoundMonster

It also includes a small set of **procedurally generated** (non-musical) sound effects created specifically for this mod using `ffmpeg` filters.

All listed clips are used as in-game SFX under `assets/kruemblegard/sounds/entity/**`.
